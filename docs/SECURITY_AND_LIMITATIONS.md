# Security and limitations

- **Unofficial API** — Albert Heijn may change or block access at any time.
- **Tokens** — Stored in the Home Assistant config entry; protect backups and diagnostics.
- **No checkout** — Cart/order changes are experimental; placing orders is not supported.
- **Rate limits** — Unknown; use reasonable polling intervals (default 300s).
- **AGPL** — appie-go is reference only; this integration is separate Python code.

Do not share appie `config.json`, authorization codes, or HA config entry exports publicly.
